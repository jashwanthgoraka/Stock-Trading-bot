---

## 🧠 Strategy: Golden Cross / Death Cross

The bot uses **Simple Moving Averages (SMA)** to detect momentum shifts:

| Signal | Condition | Action |
|--------|-----------|--------|
| 📈 **Golden Cross** | 10-day SMA crosses **above** 50-day SMA | **BUY** (long) |
| 📉 **Death Cross** | 10-day SMA crosses **below** 50-day SMA | **SELL** (short) |
| 🛑 **Stop-Loss** | Daily change exceeds ±10% | **Auto-close** position |

```
     Price
      │         ╭── Fast MA (10d)
      │    ╭────╯
      │   ╱  ◄── Golden Cross (BUY)
      │──╳────────────────────────────▶ Time
      │   ╲  ◄── Death Cross (SELL)
      │    ╰────╮
      │         ╰── Slow MA (50d)
```

## ⚡ Quick Start

### 1. Clone the Repository

```bash
git clone https://github.com/jashwanthgoraka/Stock-Trading-bot.git
cd Stock-Trading-bot
```

### 2. Create a Virtual Environment

```bash
python -m venv venv
source venv/bin/activate       # macOS/Linux
venv\Scripts\activate          # Windows
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure API Credentials

```bash
cp .env.example .env
```

Edit `.env` and add your **Alpaca paper trading** credentials:

```env
ALPACA_API_KEY=your_paper_api_key_here
ALPACA_SECRET_KEY=your_secret_key_here
ALPACA_BASE_URL=https://paper-api.alpaca.markets
```

> Get free paper trading credentials at [alpaca.markets](https://alpaca.markets)

### 5. Initialize S&P 500 Data

```bash
python setup.py
```

This downloads the current S&P 500 ticker list into `data/`.

### 6. Run the Bot

```bash
python main.py
```

## ⚙️ Configuration

All parameters are in `config/settings.py`:

| Parameter | Default | Description |
|-----------|---------|-------------|
| `FAST_MA_WINDOW` | `10` | Short-term moving average window (days) |
| `SLOW_MA_WINDOW` | `50` | Long-term moving average window (days) |
| `STOP_LOSS_THRESHOLD` | `0.10` | Max daily loss before auto-close (10%) |
| `TICKER_LIMIT` | `200` | Number of S&P 500 stocks to monitor |
| `LOOP_INTERVAL_SECONDS` | `3600` | Time between trading cycles (1 hour) |

## 🔒 Security Notes

- API keys are **never hardcoded** — loaded from `.env` via `python-dotenv`
- `.env` and `secret.py` are listed in `.gitignore` and will never be committed
- The bot uses **paper trading** by default — no real money at risk
